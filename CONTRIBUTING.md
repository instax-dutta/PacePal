# Contributing to PacePal

We welcome contributions to PacePal! Here's how you can help:

## Ways to Contribute

-   **Report Bugs:** Help us identify and fix issues by reporting them on our [issue tracker](https://github.com/instax-dutta/PacePal/issues).
-   **Suggest Features:** Share your ideas for new features and improvements.
-   **Submit Code:** Contribute code by submitting pull requests.

## Contribution Guidelines

1.  Fork the repository.
2.  Create a new branch for your feature or bug fix.
3.  Write clear, concise, and well-documented code.
4.  Test your changes thoroughly.
5.  Submit a pull request with a detailed description of your changes.

## Code Style

Please follow the existing code style and conventions.

Thank you for contributing to PacePal!
